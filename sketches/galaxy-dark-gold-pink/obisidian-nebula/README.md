## Variant: Obsidian Nebula

### Design stance

Premium, lisible et cosmique sans sacrifier la densité du cockpit.

### Key choices

- Layout: cockpit dense, rail + grille opérationnelle
- Color: noir, or, rose galaxie
- Interaction: navigation, focus mode, question modèle

### Trade-offs

Équilibre le plus sûr pour un usage quotidien.

### Best for

Autowin OS sur écran desktop.
